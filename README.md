# Github 锚点

- 1~6 级标题添加同名锚点链接

  ```markdown
  [标题1](#标题1)
  [标题2](#标题2)
  [标题3](#标题3)
  
  # 标题1
  ## 标题2
  ### 标题3
  ```

- 大写字母要小写

  ```markdown
  [Github标题1](#github标题1)
  
  ### Github标题1
  ```

- 多单词之间空格用 `-` 代替

  ```markdown
  [Github 标题2 Test](#github-标题2-test)
  
  ### Github 标题2 Test
  ```

- 多级序号要去除 `.`

  ```markdown
  [2.3. Github 标题](#23-github-标题)
  
  ### 2.3. Github 标题
  ```
